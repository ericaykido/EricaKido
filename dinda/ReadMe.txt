Para se usar o programa, criar um programa em java em algum compilador java (NetBeans, Eclipse, outros) ou apenas salvar
a pasta git em algum diretório. 
Se for o caso do primeiro, colocar os arquivos contas.csv e transacaoes.csv para o mesmo diretório que foi criado o 
programa, após apenas executar o programa pelo compilador e o arquivo com as respostas finais irá ser criado 
automaticamente na pasta que o programa foi criado.
Se for o caso do segundo, abrir a linha de comando e ir ao diretório que foi salvo o programa, após isso executar as 
linhas de comando javac Dinda.java e java Dinda.java. Deixando os arquivos contas.csv e transacoes.csv no mesmo diretorio.
Após abrir o arquivo criado final.csv para exibir o resultado obtido.

Não esquecer de deixar todos os arquivos no mesmo diretório!!
